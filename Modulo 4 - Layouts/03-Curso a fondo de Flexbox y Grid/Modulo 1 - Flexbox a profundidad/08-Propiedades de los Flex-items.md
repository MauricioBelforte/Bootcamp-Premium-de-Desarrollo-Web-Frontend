# order

Permite cambiar el orden de los elementos
Por defecto `order:0;`. Mientras mas bajo mas adelante en el eje principal.

Ejemplos:
`order:10;`
`order:-10;`
`order:100;`


**Caso practico podria usarse en media queries para cambiar el orden si cambia el tama√±o de pantalla.**
